# 🛒 TPV Proyect - Sistema de Gestión de Ventas (Vanilla PHP)

![PHP](https://img.shields.io/badge/Language-PHP%20Native-777bb4?logo=php)
![MySQL](https://img.shields.io/badge/DB-MySQL-4479a1?logo=mysql)
![Security](https://img.shields.io/badge/Auth-Session%20Based-orange)

**TPV Proyect** es una solución integral de Terminal Punto de Venta desarrollada en **PHP nativo**. El enfoque principal de este proyecto es demostrar el dominio de la lógica de servidor, la gestión de sesiones y la persistencia de datos sin dependencias de frameworks externos.

## 🛠️ Stack Tecnológico y Conceptos

* **Backend:** PHP Nativo (Vanilla PHP).
* **Gestión de Sesiones:** Implementación de `$_SESSION` para el control de acceso y persistencia de estados de usuario.
* **Base de Datos:** MySQL con integración mediante **PDO** o **MySQLi** para consultas seguras.
* **Frontend:** HTML5, CSS3 y JavaScript para la interactividad del punto de venta.

## 🚀 Funcionalidades Implementadas

1.  **Sistema CRUD Completo:** Gestión total del catálogo de productos (Alta, Baja, Modificación y Consulta).
2.  **Autenticación de Usuarios:** Control de acceso seguro mediante manejo de sesiones de PHP.
3.  **Lógica de Carrito/Venta:** Procesamiento de transacciones calculando totales, impuestos y stock en tiempo real.
4.  **Arquitectura Limpia:** Organización de código enfocada en la separación de lógica de negocio y presentación.

---



## ⚙️ Configuración Local

1.  Clonar el repositorio.
2.  Importar el archivo `.sql` (si lo tienes) en tu servidor MySQL.
3.  Configurar la conexión en el archivo correspondiente (ej: `db.php` o `config.php`).
4.  Ejecutar en un servidor local (XAMPP/WampServer).
